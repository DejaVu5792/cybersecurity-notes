# Kali Linux - File Navigation, Manipulation, & Mode Change

## Objective
Perform core file system operations using the terminal.

## Operations Performed
- Checked current working directory using pwd
- Listed files and their permissions using ls/ls -la
- Made directories using mkdir
- Navigated across directories using cd
- Created text files using touch
- Wrote contents using echo and redirection (>)
- Viewed file contents using cat
- Renamed files using mv
- Copied files using cp
- Deleted files using rm with confirmation (-i)
- Created a shared folder between Kali Linux and my original OS

## Observations
- Redirection (>) overwrites a file
- Using rm with -i flag prevents accidental deletion

## Outcome
Successfully performed the operations without looking at my notes.
